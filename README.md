
This project uses Allegiant's 2024 financial report data to forecast operating performance for the years 2025 through 2030.
The Data sheet possesses the raw operating revenue and operating expense data from the report, as well as calculated averages
for each operating revenue and expense category. At the top the user can change values for fare, inflation, fuel cost per gallon,
and number of aircraft leased. 
The Income Statement includes the forecasted operating revenue and expense values, as well as the operating profit values for each year.
Forecasted values are mostly determined through historical growth rates. Passenger earnings are the product of the forecasted number of passengers and the fare price set in the Data sheet.
Third party and resort revenues are calculated by multiplying the number of passengers by the third party revenue per passenger and resort revenue
per passenger respectively.
Aircraft fuel expenses are determined by the cost of fuel per gallon and the amount of gallons consumed. This model assumes a flat aircraft lease rental expense, based upon the values for the years 2022 through 2024.
Special charge expenses are determined by a random number generator to reflect the volatility of the expense category.
The Insights sheet provides the maximum and minimum values for forecasted profit and forecasted special charges, and includes two 
line charts that display the different values for each year.
For future developments, I would like to add more data visualizations and construct pivot tables to better understand the model.
